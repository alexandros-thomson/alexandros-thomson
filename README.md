## 🏃‍♂️ Local Testing with Act

This repository supports local GitHub Actions testing using [nektos/act](https://github.com/nektos/act):

```bash
# Quick start - list available jobs
./act-run.sh --list

# Run specific job
./act-run.sh -j test          # Run test job from main workflow
./act-run.sh -j lint          # Run lint job from main workflow

# Run with specific workflow
./act-run.sh -j test -w .github/workflows/deno.yml

# Dry run (no actual execution)
./act-run.sh -j test --dry-run
```
